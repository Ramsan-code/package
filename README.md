# Premium Pricing Table Template

This is a modern, high-conversion pricing table template inspired by premium digital service layouts.

## Features
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **Premium Aesthetics**: Dark theme with vibrant accent colors, glassmorphism-inspired cards, and smooth hover effects.
- **SEO Optimized**: Semantic HTML5 structure with proper heading hierarchy and meta tags.
- **Easy Customization**: Built with CSS variables for quick color and theme changes.
- **Smooth Animations**: Intersection observer-powered entrance animations and interactive hover states.

## Customization Guide

### Changing Colors
All main colors are defined at the top of `style.css` in the `:root` section. To change the accent color of a specific card, modify the inline `--accent-color` variable in `index.html`.

### Adding Features
Simply add more `<li>` elements to the `features` list in `index.html`. The grid will automatically adjust.

### Integrating with Your Site
You can copy the contents of `style.css` into your main stylesheet and the HTML structure into your component or page.
